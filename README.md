# Тавлеи

## Правила игры

Игра предназначена для двух игроков.  

Поле выглядит следующим образом:

```
X-------X
---------
---------
---------
----T----
---------
---------
---------
X-------X
```

Обозначения:
* `X` - выход
* `T` - трон
* `-` - простая ячейка

В начале игры фигуры на поле расставлены следующим образом:

```
---AAA---
----A----
----D----
A---D---A
AADDKDDAA
A---D---A
----D----
----A----
---AAA---
```

Обозначения:
* `A` - нападающий
* `D` - защитник
* `K` - князь
* `-` - нет фигуры

Один игрок занимает сторону нападающих (`A`), другой - защитников (`D`) и князя (`K`).  
Нападающие, чтобы победить, должны захватить князя (о том, как захватывать фигуры, будет сказано ниже).  
Для победы защитников князь должен достичь одного из четырёх выходов.  
Игроки совершают ходы по очереди. Первыми ходят нападающие.  
В свой ход игрок должен передвинуть одну любую свою фигуру на произвольное число свободных клеток по горизонтали или вертикали. "Перескакивать" через стоящие на пути фигуры нельзя.  
Князь, в отличие от всех остальных фишек, может быть перемещён не более чем на три клетки. Но только он может вставать на трон и на выходы – остальные фигуры на эти клетки ставить нельзя (переносить фишки через трон можно).  
Фигура считается захваченной и убирается с игрового поля, если в результате хода противника (но не в результате хода владельца фигуры!) она оказалась:

1. Зажата между двумя вражескими фигурами

```
---
ADA
---
```

2. Зажата между вражеской фигурой и выходом (князя это правило тоже касается)

```
XAD
---
```

3. Нападающий зажат между князем или защитником и троном

```
---
TAD
---
```

4. Защитник зажат между нападающим и пустым троном

```
---
TDA
---
```

Фигура должна быть зажата строго по горизонтали или вертикали – зажимать «углом», как показано на рисунке, нельзя:

```
DA-
A--
---
```

Князя можно захватить так же, как и все остальные фигуры, кроме следующих случаев:

1. Князь находится на троне.
В этом случае нападающие должны занять все
четыре клетки вокруг трона, перекрыв князю
все пути к отступлению:

```
-A-
AKA
-A-
```

2. Князь находится на соседней с троном клетке по горизонтали или вертикали. В этом случае нападающие должны занять все три клетки вокруг князя, прижав его к трону:

```
-A-
AKT
-A-
```

В поле князь участвует в сражении также, как и другие фигуры.
В случае, если соперник не может сделать никакой ход (все фигуры заблокированы), он признается проигравшим.

Итак, приготовьтесь к бою!
И да хранит вас ваш оберег!

Если не осталось ходов, игрок проиграл.
